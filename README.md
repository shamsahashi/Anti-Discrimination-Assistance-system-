
# Anti-Discrimination Assistance System (ADAS)

## Overview

The Anti-Discrimination Assistance System (ADAS) is a web-based application designed to help users report and document instances of discrimination in a structured and accessible way. The system allows individuals to submit detailed reports and supporting evidence, while providing administrators with tools to review and manage those submissions.

## Purpose

Discrimination is often underreported due to lack of accessible reporting tools or uncertainty about how cases will be handled. ADAS addresses this by offering a centralized platform where users can document incidents clearly and securely, helping promote accountability and better case tracking.

## Features

### User Functionality

* Submit discrimination reports through a structured form
* Upload supporting evidence such as screenshots or documents
* Receive confirmation after submission

### Admin Functionality

* Access a dashboard to view submitted reports
* Review report details and uploaded evidence
* Manage and track cases

## Installation and Setup

1. Clone the repository

```
git clone https://github.com/your-username/ADAS.git
cd ADAS
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

4. Open in browser

```
http://127.0.0.1:5000/
```
